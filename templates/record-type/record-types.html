Record Types

There are two metadata fields required and used by Pythia for state change calculation:

  • Context: Describes the context in which the fact was recorded. For example 'TestCase1'.
  • SeqNum: The ordinal number of the fact recorded. For example, the first fact has SeqNum = 0 and the second SeqNum = 1, and so forth.
